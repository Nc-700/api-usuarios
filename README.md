# 🚀 API de Usuários - Node.js

API REST desenvolvida com Node.js e Express para gerenciamento de usuários, com integração a banco de dados SQLite.

---

## 📌 Funcionalidades
- ✅ Listar usuários
- ✅ Criar usuário
- ✅ Atualizar usuário
- ✅ Deletar usuário

---

## 🛠️ Tecnologias utilizadas
- Node.js
- Express
- SQLite
- JavaScript

---

## ▶️ Como rodar o projeto

```bash
npm install
npm start
